# Fonts

`outfit-latin.woff2` / `outfit-latin-ext.woff2` — [Outfit](https://fonts.google.com/specimen/Outfit)
variable font (weights 400–900), latin and latin-ext subsets, self-hosted so the
app stays dependency-free and works offline.

Licensed under the [SIL Open Font License 1.1](https://openfontlicense.org/).
Copyright 2021 The Outfit Project Authors (https://github.com/Outfitio/Outfit-Fonts).

Used for Latin display text (titles, buttons, labels) via `--font-display` in
`style.css`; CJK and other scripts fall back to the system stack (`--font-ui`).
